# Install

aqua is a single binary written in Go.

1. Install the binary `aqua` in `PATH`
1. add `~/.aqua/bin` to the environment variable `PATH`

## Homebrew

```console
$ brew install aquaproj/aqua/aqua
$ export PATH=$HOME/.aqua/bin:$PATH
```
